# Test Spec — 최근 내역 요약 Export 기능

- 작성일: 2026-02-28
- 대상: `src/components/TransactionTable.tsx`

## 수동 테스트 케이스

### 1) 버튼 노출/스타일
- 조건: 대시보드 > 최근 내역 요약 진입
- 기대값:
  - Export 버튼이 헤더 우측에 보인다.
  - 반투명/blur 스타일이며 기존 컴포넌트 톤과 일관된다.

### 2) 데이터 있을 때 Export 성공
- 조건: 거래 데이터 1건 이상 존재
- 액션: Export 버튼 클릭
- 기대값:
  - `transactions-YYYY-MM-export.csv` 파일이 다운로드된다.
  - CSV 첫 줄은 헤더이고, 데이터 행 수는 현재 필터 결과 건수와 동일하다.
  - 한글 텍스트(메모/카테고리)가 깨지지 않는다.

### 3) 필터 반영 검증
- 조건: 검색/카테고리/결제수단/실적 필터 조합 적용
- 액션: Export 버튼 클릭
- 기대값:
  - 다운로드된 CSV에는 현재 화면에 표시된 필터 결과만 포함된다.

### 4) 데이터 없을 때 상태
- 조건: 필터 결과 0건
- 기대값:
  - Export 버튼이 비활성화된다.
  - 클릭해도 다운로드가 시작되지 않는다.

### 5) 회귀 체크
- 범위:
  - 거래 추가/삭제
  - 필터 변경
  - 모바일/데스크톱 리스트 표시
- 기대값:
  - 기존 동작이 유지된다.
