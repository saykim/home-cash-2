# PRD — 최근 내역 요약 Export 기능

- 작성일: 2026-02-28
- 범위: `최근 내역 요약` 섹션 UI + 클라이언트 내 추출(Export) 기능

## 배경
사용자가 월별/필터링된 입력 거래 데이터를 외부에서 재활용할 수 있도록 빠른 추출 기능이 필요함.

## 목표
1. 최근 내역 요약 섹션 상단에 **Export 버튼**을 추가한다.
2. 현재 화면 필터가 반영된 거래 목록을 CSV로 내려받을 수 있게 한다.
3. 기존 대시보드 디자인 톤을 유지하고, 버튼은 반투명/세련/일관된 스타일을 적용한다.

## 비목표
- 서버 API 추가/변경
- 파일 포맷 다중화(XLSX, PDF)
- 백그라운드 대용량 내보내기

## 기능 요구사항
1. Export 버튼은 최근 내역 요약 헤더 우측(검색 기준 안내와 동일 그룹)에 위치한다.
2. 클릭 시 현재 `transactions`(필터 적용 결과)를 CSV 파일로 다운로드한다.
3. CSV 컬럼은 아래를 포함한다.
   - 거래일
   - 구분
   - 내역
   - 결제수단
   - 금액
   - 실적반영
   - 청구제외
4. 파일명은 월 기준 정보가 포함된 형태(예: `transactions-2026-02-export.csv`)를 사용한다.
5. 추출할 데이터가 없으면 버튼은 비활성화한다.

## UX/디자인 요구사항
- 기존 카드/칩 스타일과 이질감 없는 형태
- 반투명 배경 + 은은한 blur 느낌
- hover/focus-visible 상태 제공(접근성)

## 성공 기준
- 최근 내역 요약 영역에서 Export 버튼이 노출되고 동작한다.
- CSV가 UTF-8 BOM 포함으로 한글이 깨지지 않는다.
- 기존 검색/필터/삭제/추가 기능에 회귀 이슈가 없다.
